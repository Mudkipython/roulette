# Roulette Lab v7：物理模拟重构

## 核心变化

v7 删除了旧版所有预设轨迹、正弦回弹和“最后强制对准号码”的逻辑。3D 模式下，开奖结果现在来自小球相对旋转转盘的最终稳定位置。

### 物理阶段

1. **外沿运动**：小球在固定外轨顺时针滚动，角速度受到滚动与空气阻力影响。
2. **临界离轨**：当向心支撑不足以抵消斜面重力分量时，自动关闭下注并离开外轨。
3. **斜面内滑**：使用极坐标方程逐步积分径向速度与角速度。
4. **挡板碰撞**：使用扫掠检测和碰撞冲量，加入切向摩擦与低恢复系数；没有人工正弦跳动。
5. **号码环**：小球进入旋转号码环后，与隔片发生相对运动碰撞并损失能量。
6. **稳定开奖**：径向、相对切向和垂直速度同时降低后，根据相对角度确定号码。

整个模拟使用 **240 Hz 固定时间步长**，避免帧率变化改变运动结果。

## 验证

- 100 轮标准物理回归
- 1,000 轮扩展物理回归
- 所有轮次自然稳定，无飞出、穿心或超时强制开奖
- 所有轮次均与号码隔片发生碰撞
- 1,000 轮覆盖全部 37 个欧式号码
- 结果均匀性卡方检验未发现显著标签偏置
- 中英法切换、科普章节路由和 390 px 移动端横向溢出检查通过
- WebGL 不可用时的 2D 回退检查通过

## 重要边界

这是针对浏览器展示优化的物理模型，不是对某一台商业轮盘进行尺寸、材料、空气阻力和机械误差校准后的数字孪生，也不能用于预测赌场结果。
