# v4 修复说明

本版针对上一版的实际问题进行了结构性重写，而不是继续调整动画参数。

## 1. 固定视角，不再颤动

- 完全删除 `OrbitControls`。
- 相机只在初始化时设置一次位置和朝向。
- 删除输局震动、镜头跟随、阻尼和自动缩放。
- 窗口尺寸变化时只更新宽高比，不移动相机。

## 2. 修正号码槽几何体

上一版扇形槽使用了错误的几何平面，视觉上像红黑各占半边。现在：

- 每个号码槽都是水平放置的独立环形扇区。
- 按欧式或美式轮盘的官方固定顺序生成。
- 红、黑、绿色由号码本身决定，而不是按位置简单交替或分区。
- 号码、分隔片和内外挡圈都跟随转子一起逆时针旋转。

## 3. 修正小球坐标和落袋位置

- 小球现在是轮盘机器局部坐标系的子对象。
- 外沿轨道、小球和号码槽共用一致的高度基准。
- 小球最终半径固定在号码槽环约 `4.47`，而不是中心轴。
- 最终落点使用正在旋转的转子相对坐标计算。

## 4. 自动连续开局

- 页面载入后自动发球，无需每局点击“发球”。
- 小球沿固定外轨顺时针转动时开放下注。
- 小球开始向内滑落的瞬间锁定下注。
- 开奖展示结束后自动进入下一局。
- 仅保留“本局后暂停 / 继续自动开局”控制。

## 5. 多注与撤销

- 可同时押多个单号和多个外围项目。
- 每次点击增加当前选中的筹码金额。
- 支持撤销最后一枚筹码。
- 支持逐项删除某一下注。
- 支持清空本局全部下注。
- 支持一键重复上一局下注组合。
- 同一局可有多个中奖下注，按各自赔率分别结算。
