# Roulette Lab v8 更新说明

## 新功能

- 增加受限 OrbitControls 镜头：拖动环绕、滚轮/双指缩放、一键复位。
- 增加独立暂停页面：继续、重置模拟、恢复视角、声音设置。
- 增加浏览器本地合成的赌场环境声与原创休闲乐垫。
- 音效、环境声可分别开关，并支持总音量调节。
- 语言选择器使用国旗加语言原名：🇨🇳 中文、🇬🇧 English、🇫🇷 Français。
- README 调整为完整中文介绍后用分割线接完整英文介绍。
- 优化中英文字体栈、标题字重、间距与移动端暂停布局。

## 镜头边界

- 禁用平移，避免轮盘离开画面中心。
- 限制最小与最大镜头距离。
- 限制俯仰角和水平环绕角度。
- 号码点击使用移动距离阈值区分点击与拖动，避免调整镜头时误下注。

## 声音说明

环境声不加载外部歌曲，全部由 Web Audio API 本地生成，包括房间底噪、远处人群纹理、筹码细节和原创和弦乐垫。浏览器首次交互后才会启动声音，以符合自动播放限制。
